# 💰 FinRelief AI

## AI-Powered Debt Settlement & Financial Analysis System

FinRelief AI is a web application that helps users manage their loans, analyze their financial health, and receive AI-powered debt settlement recommendations. The platform provides a secure, user-friendly interface for tracking loans and improving financial decision-making.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 💳 Loan Management (Add, Edit, Delete, View)
- 📊 Dashboard with Charts
- 📈 Financial Health Analysis
- 🤖 AI-Based Debt Settlement Recommendations
- 👤 User Profile Management
- 📱 Responsive User Interface

---


### Frontend

```bash
npm install
npm run dev
```
